### Obj files
Son archivos que definen cualidades de un objeto, como son las cordenadas de los vertices o
los indices de cada vertice(empezando en 1) por cara. 
| Format | Meaning |
| -------------- | --------------- |
| vN | Vertex index |
| vtN | Vertex Texture index |
| vnN | Vertex Normal index |
| f | Vertex of a face |
